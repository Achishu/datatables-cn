## 第一步：引入下面两个文件

css:
```html
//cdn.datatables.net/1.10.16/css/jquery.dataTables.min.css
```

js:
```html
//cdn.datatables.net/1.10.16/js/jquery.dataTables.min.js
```

## 第二步：在页面上执行下面代码

```javascript
$(document).ready( function () {
    $('#myTable').DataTable();
} );
```
